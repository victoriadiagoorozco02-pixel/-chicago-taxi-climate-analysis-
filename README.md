# Taxi Trip Analysis - Weather Impact Study

## Project Description
This project analyzes taxi trip data to evaluate how weather conditions impact trip duration.

The study includes statistical testing to validate whether differences observed are significant.

---

## Objective
Determine whether weather conditions significantly affect taxi trip duration.

---

## Dataset
The dataset includes:
- Trip duration  
- Weather conditions  
- Dates and trip details  

---

## Tools Used
- Python (Pandas)  
- Matplotlib / Seaborn  
- SciPy  

---

## Analysis Process

1. Data cleaning  
2. Exploratory data analysis  
3. Hypothesis testing  
4. Statistical validation  

---

## Key Insights

- There is a statistically significant difference in trip duration between good and bad weather conditions.  
- Trips on rainy Saturdays last approximately 7 minutes longer on average.  

---

## Recommendations

- Adjust operational planning during adverse weather conditions  
- Improve demand forecasting considering weather impact  

---

## Conclusion

Weather conditions have a measurable impact on trip duration, highlighting the importance of external factors in operational performance.

---

## Next Steps
